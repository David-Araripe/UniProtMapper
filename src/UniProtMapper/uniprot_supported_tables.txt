<div><h2 id="uniprotkb-column-names-for-programmatic-access">UniProtKB column names for programmatic access</h2>
<p>This document lists the differences between the returned columns by RESTful APIs.
User can ask required columns returned by an API by passing the <em>Returned Field</em> in the request url
and the response will have those requested fields.</p>
<p>A specific page listing cross-reference fields can be found here: <a href="/help/return_fields_databases">Return Fields Databases</a>.</p>
<p>For example, to get UniProtKB entry with accession and proteomes in new API run the below <code>curl</code> command:</p>
<pre><code>curl https://rest.uniprot.org/uniprotkb/search?query=human&amp;fields=accession,xref_proteomes
</code></pre>
<p>The <code>Label</code> is the readable name of the returned field. The <code>Label</code> is shown on the website/TSV or in Excel format.</p>
